Global Energy Savings from Using Dark Mode

This analysis assumes every phone in the world has an OLED display, is used at 100% brightness, and goes from 100% to 0% battery daily.

If everyone in the world used dark mode on their phones, the amount of energy saved would be massive. According to research from Purdue University, dark mode saves around 43% of power at 100% brightness on OLED displays.

E(Wh) = Q(mAh) × V(V) / 1000 4,500 mAh × 3.7 V / 1000 = 16.65 Wh Energy Saved = 16.65 × 0.43 = 7.1596 Wh

The average phone battery holds 16.65 Wh. Saving 43% of that is 7.1596 Wh saved per phone per day.

Population-Level Impact

  • World population: 8,057,277,567 people
  • Total energy saved per day: 8,057,277,567 × 7.1596 Wh = 57,701,203,668.7 Wh
  • Daily savings in watts: 57,701,203,668.7 Wh ÷ 24 = 2,404,216,819.53 W

Converting to Gigawatts and Annual Savings

Daily gigawatt-hours saved: 57.70 GWh

Daily gigawatts saved: 2.404 GW

  • Annual gigawatts saved: 2.404 × 365 = 877.54 GW
  • Annual gigawatt-hours saved: 57.701 × 365 = 21,061 GWh

This is about the same as 878 trillion watts or 21,061 trillion Wh saved each year.

Debugging Energy Math

const battery_mAh = 4500; const voltage = 3.7; const daily_users = 8057277567; const dark_mode_saving = 0.43; const energy_per_phone = (battery_mAh * voltage) / 1000; // 16.65 Wh const saved_energy = energy_per_phone * dark_mode_saving; // 7.1596 Wh const total_saved = saved_energy * daily_users; const daily_gWh = total_saved / 1e9; const yearly_gWh = daily_gWh * 365; console.log(yearly_gWh); // ~21,061 GWh
